# Product Specs

## Продукт

"Визитка" — конструктор персональной страницы-визитки. Пользователь регистрируется, получает публичный URL `/:username` и наполняет его блоками контента в адаптивной бенто-сетке.

## Целевой пользователь

Специалист или творческий человек, которому нужна простая link-in-bio страница без написания кода. Добавляет блоки для ссылок, фото, соцсетей, карты, заметок и делится URL.

## Ограничения продукта

- Один профиль на аккаунт.
- Публичные страницы доступны без авторизации.
- Редактор — одиночный (нет совместного редактирования).
- Без кастомных доменов — URL всегда `/:username` на домене приложения.
